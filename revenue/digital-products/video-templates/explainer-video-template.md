## 8. Premium Pro Tips (Bonus — Adds Real Value)

1. **The 3-Second Rule:** If the hook text doesn't appear within the first 2.5 seconds of render, viewers scroll past. Remotion skips 0.5s of black leader by default — set `delayRenderTimeout` to 100ms max at frame 0.
2. **Subtitle Burn-In:** Always burn subtitles for silent-auto-play on social. 85% of explainer views happen without sound.
3. **Thumbnail Frame:** The CTA scene works best as the manual thumbnail frame. Render the last frame as `thumbnail.png` using Remotion's `getStillOnPort()` for the VideoHive preview.
4. **A/B Hook Testing:** Generate 3 versions of Scene 1 (different hook angles) and use the AVG pipeline's `sceneVariants` param to batch-render all 3 in one command. Keep the variant with the highest watch-through rate.
5. **Audio Ducking Precision:** Duck music by exactly -14dB when VO is present; no ducking on silent beats. This prevents the "pumping" effect that sounds amateur.
6. **Subtle Motion:** Add a 0.5% scale keyframe (100% → 100.5%) over 60s to every static scene. It creates a barely-perceptible "breathing" effect that holds viewer attention.
