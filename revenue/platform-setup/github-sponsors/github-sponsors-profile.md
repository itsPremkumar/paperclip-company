## Landing Page Description (Visible to All Visitors)

### Who I Am

I'm Prem Kumar, founder of Prem Autonomous Co — a business built on and operated by autonomous AI agents. I build open-source tools, automation frameworks, and prompt libraries that help developers and solopreneurs leverage AI agents for real work.

I've been writing software since 2019 and operating agent-driven systems since early 2024. My company runs on Paperclip and Hermes, orchestrated by 7 specialized agents that research, write, code, review, and publish — handling the volume so I can focus on direction.

### What I Build

**Open-source tools for the agent economy.** My repositories focus on three areas:

- **Paperclip + Hermes configurations and extensions** — production-ready agent profiles, MCP server configs, and workflow templates that make multi-agent setups faster to deploy.
- **CLI tools for business automation** — scripts and utilities for content management, research aggregation, code review pipelines, and terminal-based business operations. No GUI, no subscriptions, no overhead.
- **Prompt engineering frameworks** — structured prompt libraries, AGENTS.md generators, and evaluation tooling that help developers get consistent, high-quality output from any LLM.

Every open-source project I publish is actively used in production at Prem Autonomous Co. If it's in my GitHub, it's earning its keep.

### Why Sponsorship Matters

I believe open-source infrastructure for the agent economy should be free and accessible. But building it takes time — time spent designing architectures, writing documentation, responding to issues, and releasing updates.

Your sponsorship directly funds:

- **More open-source releases** — every sponsor milestone unlocks a new tool or framework I release publicly
- **Documentation and guides** — I write detailed READMEs, setup guides, and architecture explainers for every project
- **Community support** — issue responses, PR reviews, and feature requests from sponsors are prioritized
- **Experimentation** — I maintain test infrastructure for multi-model, multi-platform compatibility

I don't run ads, sell user data, or gate features behind paywalls. Sponsorship is the cleanest way to support the work if you find it valuable.
